# Hikmah IT Website - Vite React Version

Professional landing website for Hikmah IT built with Vite, React and custom CSS.

## Main Features

- Professional homepage redesign
- E-commerce website with admin panel section
- Madrasah Management System section
- Service, pricing, portfolio, FAQ and contact sections
- Responsive navbar and mobile layout
- Brand colors: navy, blue, amber, emerald
- Clean reusable React components
- SEO-ready `index.html`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Main Files

- `src/App.jsx` - Main app layout
- `src/data/siteData.js` - All content/data
- `src/components/` - Reusable UI components
- `src/sections/` - Website sections
- `src/styles.css` - Full styling and responsive design
- `public/` - Existing images/assets copied from old project
